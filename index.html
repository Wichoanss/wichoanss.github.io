Website Empresarial para Registro de Datos y Métricos

Este proyecto es un sitio web empresarial diseñado para registrar y gestionar todo tipo de datos y métricos. La plataforma permite a los usuarios iniciar sesión de manera segura, acceder a diversas aplicaciones para el manejo de datos, y visualizar métricas en tiempo real a través de gráficos interactivos. Las características incluyen:

  • Sistema de Autenticación: Inicio de sesión seguro con recuperación de contraseñas.
  • Gestión de Datos: Aplicaciones específicas para diferentes tipos de datos, incluyendo control de temperatura, validación de herramientas y más.
  • Visualización de Métricas: Gráficos interactivos que muestran datos en tiempo real, facilitando el monitoreo y análisis.
